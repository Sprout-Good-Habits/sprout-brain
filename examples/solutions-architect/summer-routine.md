# Example: Summer Routine

Last verified: 2026-06-02

## User prompt

```text
I want a summer day plan for my 6-year-old.
```

## Route

Program planner.

## Recommended Sprout shape

- Morning conversation check-in.
- One learning canvas.
- One parent-reviewed offline task.
- One creative journal or drawing activity.
- Reward progress check.

## Good current plan

```text
Morning: Sprout asks what kind of day the kid wants.
Learning: 10-minute canvas activity.
Outside: parent-reviewed offline mission.
Creative: journal together or draw something.
Reward: show progress toward one parent-approved reward.
```

## Avoid

- Camera proof of outdoor activity.
- Complex all-day adaptive scheduler unless current product supports it.
