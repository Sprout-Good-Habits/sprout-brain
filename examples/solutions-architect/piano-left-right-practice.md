# Example: Piano Left/right Hand Practice

Last verified: 2026-06-02

## User prompt

```text
My kid confuses left hand and right hand when practicing music. Can Sprout
help?
```

## Route

Simple family activity.

## Recommended Sprout shape

- Canvas mini-game or conversation practice.
- Ask the child to choose left/right based on prompts.
- Reward in-canvas completion or score if it is a real in-canvas practice.
- Parent-reviewed claim only if the parent wants to reward real piano practice
  outside Sprout.

## Good current plan

```text
Make a short left/right practice game. Sprout shows a hand prompt, the child
taps left or right, and the canvas scores the round. This practices the concept
inside Sprout.
```

## Avoid

```text
Sprout watches the kid play piano and verifies finger placement.
```
