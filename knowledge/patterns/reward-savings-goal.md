# Pattern: Reward Savings Goal

Use when the child wants a concrete item and the parent wants effort to
accumulate toward it.

Last verified: 2026-06-02

## Current Sprout shape

- `reward_create` for the item.
- Child assignment sets the gem requirement.
- Tasks/reviews award gems.
- Child claims the reward.
- Parent approves the claim.

## Example

- "Switch game" at 50 gems.
- Khan mastery earns gems.
- The child redeems the reward once they have enough gems.

## Avoid

- Treating the reward as the same thing as task completion.
- Auto-approving expensive or parent-sensitive redemptions by default.
