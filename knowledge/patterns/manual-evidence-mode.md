# Pattern: Manual Evidence Mode

Use when the home-agent connection is not ready but the parent still wants the
Sprout behavior now.

Last verified: 2026-06-02

## Current Sprout shape

- Parent checks the external source.
- Parent enters or pastes a summary.
- Agent suggests gems.
- Parent approves.
- Agent calls the review/write path.

## Good parent-facing wording

"This starts in manual evidence mode. You check Khan and enter what happened.
Later, if your home agent can provide the evidence automatically, we can remove
the manual entry step without changing the child's quest."

## Avoid

- Pretending manual evidence mode is automated.
- Saying Sprout can see the third-party dashboard by itself.
