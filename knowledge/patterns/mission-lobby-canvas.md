# Pattern: Mission Lobby Canvas

Use when a child needs motivation, orientation, and a sense of progress.

Last verified: 2026-06-02

## Current Sprout shape

- Canvas shows the mission, schedule, progress, and next step.
- Canvas can emit completion when the child asks for a check or finishes the
  in-canvas portion.
- Skill wraps the canvas.
- Task delivers it on a schedule.

## Good content

- Today's mission.
- Focus area.
- Reward progress.
- What counts.
- "Ask parent to check" action.

## Avoid

- Treating the canvas completion as proof of external work.
- Letting the canvas grant gems directly for outside work.
- Building a canvas that depends on external `fetch`.
