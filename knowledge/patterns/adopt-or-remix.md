# Pattern: Adopt or Remix Before Custom Build

Use when the parent wants something likely to exist in the marketplace or saw a
similar example.

Last verified: 2026-06-02

## Current Sprout shape

- Prompt the parent to search the Skills Marketplace for relevant terms.
- If a close match exists, adopt it into a private family copy.
- Personalize it for the child.
- If it becomes meaningfully different, it can be republished as a remix when
  marketplace publishing supports that path.

## Good wording

"Before we make this from scratch, search the Sprout Skills Marketplace for
Khan, math quest, or multiplication. If there is a close match, we can adopt
and personalize it. If not, I will create a child-specific version."

## Avoid

- Calling this one-click install into Codex or Claude Code.
- Publishing trivial copy changes as meaningful remixes.
