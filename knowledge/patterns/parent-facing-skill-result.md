# Pattern: Parent-facing Skill Result With Suggestions

Use when a parent wants approval from a phone or parent-facing flow, or when
external evidence needs to be summarized before reward.

Last verified: 2026-06-02

## Current Sprout shape

Produce a result that contains:

- evidence summary
- suggested gems
- reason
- approve/reject recommendation

Gems are not awarded until the parent approves.

## Example

"The child did 14 minutes of multiplication and scored 86%. Suggested reward: 4 gems
(1 consistency + 3 mastery). Approve?"

## Avoid

- Calling the suggestion an approval.
- Awarding gems before explicit parent approval.
- Promising a custom phone screen if only agent-mediated approval is available.
