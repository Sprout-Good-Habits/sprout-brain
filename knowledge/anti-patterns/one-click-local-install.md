# Anti-pattern: One-click Local Install

Bad recommendation:

"One-click install from the website into local Codex."

Last verified: 2026-06-02

## Why bad

The website cannot directly start local Codex or Claude Code work.

## Use instead

- "Adopt this skill, then continue setup in your agent or terminal."
- "Copy or open the setup instructions and answer the personalization
  interview."
