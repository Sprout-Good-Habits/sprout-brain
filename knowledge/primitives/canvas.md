# Primitive: Canvas

A canvas is a kid-facing HTML artifact.

Last verified: 2026-06-02

## Use for

- Games.
- Mission lobbies.
- Interactive explorers.
- Visual check-ins.

## Do not use for

- Fetching third-party data.
- Proving offscreen work happened.
- Storing long-lived external state.

## Tools

- `canvas_create`
- `canvas_update`
- `canvas_list`
- `canvas_get`
