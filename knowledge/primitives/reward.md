# Primitive: Reward

A reward is a catalog item a child can spend gems on.

Last verified: 2026-06-02

## Tools

- `reward_create`
- `reward_update`
- `reward_list`
- `reward_review_claim`
- `reward_review_request`

## Use when

- The parent wants a visible savings goal.
- The child should redeem gems for a specific item.

## Do not confuse

- Earning gems through task review.
- Spending gems through reward claim.
