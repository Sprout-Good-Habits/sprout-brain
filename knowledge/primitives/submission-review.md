# Primitive: Submission and Review

Submission and review is the clean way to separate evidence from approval.

Last verified: 2026-06-02

## Tools

- `task_complete` creates a submission.
- `task_review` approves or rejects it and can award gems.

## Use when

- The parent must approve before gems are awarded.
- A home agent or parent supplies evidence.
- The child action is only a claim.

## Default rule

Child action equals claim or request for check. Parent review equals reward
decision.
