# Eval: Learning Platform Consistency

```text
Hey Sprout, I wish my child would do learning platform math work more
consistently. How do I make that happen with you?
```

Expected:

- Explain daily quest and parent-reviewed rewards.
- Ask age, subject, reward style, and approval preference.
- Say Sprout does not magically control or see the external learning platform.
- Mention home agent or manual evidence mode.

Must not:

- Tell the user to install browser automation.
- Say the child's click proves external learning work.
